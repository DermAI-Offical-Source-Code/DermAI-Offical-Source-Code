# 🌟 DermAI: Multi-Platform AI System for Dermatological Assistance

Welcome to the DermAI project repository! DermAI is a multi-platform AI system designed for dermatological assistance. This project was built to explore the application of Artificial Intelligence and Computer Vision in healthcare, specifically aiming to provide accessible, early-stage skin condition analysis.

## 📊 Core Metrics & Validation
* **Dataset Scale**: The model was trained on a comprehensive dataset of approximately 270,000 images.
* **Clinical Accuracy**: Achieved an 86.1% accuracy rate during clinical validation.
* **Explainable AI**: Integrates Grad-CAM to generate heatmaps, visually highlighting the damaged skin areas the AI focuses on to make its predictions.

## 🏗️ System Architecture (Microservices)
The DermAI ecosystem is modularized into independent repositories to optimize development and deployment:
1. **[DermAI-Website]**: The central web server built with Django.
2. **[DermAI-Mobile]**: Cross-platform mobile applications developed using React Native.
3. **[DermAI-Desktop]**: A native desktop application for professional desktop environments.
4. **[DermAI-Model]**: The deep learning model training repository utilizing the ConvNeXt-Tiny architecture.
5. **[DermAI-API]**: The dedicated AI processing server built with FastAPI.
